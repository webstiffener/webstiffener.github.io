# 蒸饭

- 洗米两次（1人份米量约150ml）
- 清水浸泡（米水体积比1：1.5）
- 蒸架放入汤锅，加水至蒸架盘面处，浸泡好的米直接放上蒸架。大火使水沸腾，然后最小火蒸煮25分钟，闷5分钟，取饭盒翻转。
- **Tips：** 锅边可放一个勺，防止锅盖跳。蒸煮10分钟时检查水是否烧干。

# 炒胡萝卜
- 胡萝卜切**薄**片，热水焯一下
- 最小火热锅，放油继续热锅
- 待锅底上方烘热，入锅翻炒，加水翻炒，放盐和酱油，翻炒后加水最小火闷，试吃，停火。
